Thing Encrypter

V 0.3 - 27.09.2024

Uses the AES Algorithm to encrypt and decrypt files.
The key is stored in a .tek (Thing Encryption Key)file, make sure to keep it safe.
Decoding the files without a key takes about 2 billion years.

To get started, press the "generate"-button to generate a new .tek key.
You can now either select files from a directory or the whole directory itself and de/encrypt them.
Depending on the ammount and the size of the files the program needs to handle,
the processing time can be anywhere between half a second and a few minutes.

It is NOT RECOMMENDED to encrypt very large (5GB<) files.
